# Rowing trophy

Rowing trophy customized for Kiel University Rowing.

Edit the text in `rowing-base-uni-kiel.FCStd` using [FreeCAD](https://www.freecadweb.org/).

## Print

- Files: `blade.stl` and `rowing-base-uni-kiel-Fusion.stl`
- Size: 20%
- Material: PLA
